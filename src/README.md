🎟️ UrbanPulse Booking System
A professional Java Swing desktop application designed to manage leisure centre memberships and sports facility bookings. This project was developed as part of the CS5002 Soft Eng module to demonstrate core Software Engineering principles and Object-Oriented Design.

🚀 Key Features
User Authentication: Secure login and registration for Members and Staff.

Membership Management: Join new tiers or renew existing memberships with automatic fee calculation.

Facility Booking: Real-time booking for sports pitches with peak/off-peak pricing logic.

Payment Processing: Integrated workflow for taking payments and managing digital receipts.

Admin Dashboard: Specialised views for staff to manage maintenance, refunds, and system reports.

🏗️ Architecture: MVC Pattern
To ensure the code is maintainable and scalable, the system follows the Model-View-Controller (MVC) architectural pattern:

Model (Booking.java, UserManager.java): Handles the data logic and business rules.

View (BookingSystem.java): Manages the Graphical User Interface (GUI) and user interactions.

Controller (BookingManager.java): Acts as the interface between the UI and the data storage.

🛠️ Technology Stack
Language: Java 17+

Framework: Java Swing (GUI)

IDE: IntelliJ IDEA

Version Control: Git & GitHub

💻 How to Run
Ensure you have the Java Development Kit (JDK) installed.

Clone this repository: git clone https://github.com/awadtariq/CS5002_Soft_Eng_Project.git
